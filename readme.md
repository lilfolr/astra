# Astra (Starship Home)

A co-op family management application that gamifies household chores through a "Spaceship Survival" metaphor.

## Documentation

Comprehensive project documentation can be found in the `docs/` directory:

- [Project Overview](docs/overview.md) - Executive summary and technical architecture.
- [Project Plan](docs/projectPlan.md) - Development roadmap and phases.
- [UI/UX Design](docs/ui.md) - Visual identity and design specifications.

## Technology Stack

- **Frontend:** React Native
- **Game Engine:** `react-native-game-engine`
- **Backend:** Firebase
