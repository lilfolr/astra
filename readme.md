# Astra (Starship Home)

A co-op family management application that gamifies household chores through a "Spaceship Survival" metaphor.

## Project Structure

This project is organized as a monorepo:

- **`apps/`**: Contains the applications.
  - **`astra_mobile/`**: The Flutter mobile application.
- **`packages/`**: Shared Dart/Flutter packages and libraries.
- **`iac/`**: Infrastructure as Code (IaC) configurations.
- **`docs/`**: Project documentation and specifications.
